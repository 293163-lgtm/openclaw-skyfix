# skyfix v0.1.0

## Initial public release

`skyfix` is a self-contained OpenClaw maintenance expert for real-machine operations.

This first public release focuses on the core maintenance loop:

- install / bootstrap
- upgrade / reinstall / repair
- config and permission normalization
- gateway / LaunchAgent repair
- desktop shortcut generation
- Telegram diagnostics
- ASR / TTS / Telegram voice diagnostics
- Telegram file / media / voice delivery diagnosis

## Why this release matters

This is not just a bag of helper scripts.
It is a single maintenance orchestrator designed to reason by fault layer and verify outcomes.

In plain terms:
- it tries to determine what kind of problem you actually have
- it avoids blind reinstall-first behavior
- it reports what was fixed, what remains broken, and what layer the failure belongs to

## Public posture

This release is intentionally `v0.1.0`, not `v1.0.0`.

It is already useful and field-tested, but still early in its public product lifecycle.
That is a deliberate versioning choice, not hesitation.

## Notable current boundaries

- WeChat support is lite diagnosis only in V0.1
- self-evolution is suggestion-only, not autonomous self-modification
- issue intelligence is reserved for difficult / ambiguous failures
